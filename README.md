# Online Health Care System

An advanced web-based platform for healthcare management, supporting patients, doctors, and administrators. The system streamlines appointment booking, user management, doctor approval, medical records, and more, with a modular and scalable architecture.

---

## 🚀 Features

- **Patient Portal:** Registration, profile management, appointment booking, view medical records.
- **Doctor Portal:** Manage schedules, view patients, write prescriptions, update profiles.
- **Admin Dashboard:** User and doctor management, approval workflows, system reports, settings.
- **Authentication:** Secure login and registration for all roles.
- **Responsive UI:** Optimized for desktop and mobile devices.
- **Data Management:** Dummy data and schema for easy testing and extension.

---

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
- **Styling:** Modular CSS (by feature & role)
- **Data:** JSON files for mock data and schema
- **No backend/server required** (static web app)

---

## 📁 Folder Structure

- `index.html` — Main entry point
- `js/` — JavaScript files (feature & role-based organization)
- `css/` — CSS stylesheets (feature & role-based organization)
- `html/` — HTML pages for all modules
- `DB/` — Database schema and dummy data
- `images/` — Assets and images
- `package.json` — Project dependencies and scripts

---

## 🖥️ Usage

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Osamaelaz/OnlineHeathCareSystem.git
   ```
2. **Navigate to the project directory:**
   ```bash
   cd OnlineHeathCareSystem
   ```
3. **Install dependencies:**
   ```bash
   npm install
   ```
4. **Run the project:**
   - Open `index.html` directly in your browser, or use a local server (e.g., Live Server extension in VS Code).
