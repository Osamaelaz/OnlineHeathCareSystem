# Online Health Care System

This project is a comprehensive Online Health Care System designed to facilitate healthcare management for patients, doctors, and administrators. It provides features such as appointment booking, user management, doctor approval, medical records, and more.

## Features
- Patient registration, profile, and appointment booking
- Doctor management, scheduling, and prescriptions
- Admin dashboard for user and doctor management
- Medical records management
- Authentication for different user roles
- Responsive UI with separate modules for Admin, Doctor, and Patient

## Project Structure
- `index.html` - Main entry point
- `js/` - JavaScript source files (organized by feature and user role)
- `css/` - CSS stylesheets (organized by feature and user role)
- `html/` - HTML pages for different modules
- `DB/` - Database schema and dummy data
- `images/` - Project images and assets
- `package.json` - Project dependencies and scripts

## Getting Started
1. **Clone the repository:**
   ```bash
   git clone https://github.com/Osamaelaz/OnlineHeathCareSystem.git
   ```
2. **Navigate to the project directory:**
   ```bash
   cd OnlineHeathCareSystem
   ```
3. **Install dependencies:**
   ```bash
   npm install
   ```
4. **Run the project:**
   - Open `index.html` in your browser, or use a local server (e.g., Live Server extension in VS Code).
